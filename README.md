ECG Regression Models with Stacking

Overview
This project builds regression models to predict the TARGET variable from ECG measurements. The notebook trains several supervised learning models and evaluates their performance on training and test data.
The project also builds an ensemble model using stacking. Stacking combines predictions from multiple models to improve prediction accuracy.

Models Used
The notebook trains the following models.
    Decision Tree Regressor
    Random Forest Regressor
    Gradient Boosting Regressor
The final ensemble model uses Stacking Ensemble Learning.
Model performance is evaluated using Mean Squared Error.

Dataset
Input files: train and test

Description
    The training dataset contains feature variables and the TARGET column.
    The TARGET column represents the value that the models predict.
The training dataset is used to train models.
The test dataset is used to evaluate prediction performance.

Workflow
The notebook performs the following steps.
Load the datasets
Separate features and target variable
Train individual models
    Decision Tree Regressor
    Random Forest Regressor
    Gradient Boosting Regressor
Evaluate predictions using mean squared error
Build a stacked ensemble model
Generate predictions using the stacked model
Save predictions from all models to a single Excel file.

Output
Output file: data/output_regression.xlsx
Sheets inside the file
    DecisionTree
    RandomForest
    GradientBoosting
    Stacking
Each sheet contains the predicted TARGET values generated by the corresponding model.
